Topic of Study: Our project tracks the expansion of Roman infrastructure across Europe. We will not analyze the specific engineering physics of a single technology. Instead, we will focus on the big picture. We'll explore how imperial decision-makers used architecture and engineering to control vast territories and populations.

Form of Presentation: We will present this research as an interactive web map with an adjustable time slider that contains information on famous historical monuments and structures. Historical records are often messy and hard to visualize. This project structures that data into a clean, accessible format. Anyone can easily see which monuments existed at any given moment. By moving the slider, users can watch the Roman Empire's technological footprint physically spread across the map. We plan to cover every monument discussed in class and possibly additional ones as well.
